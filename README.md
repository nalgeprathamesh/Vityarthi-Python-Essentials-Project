A simple, classic **Rock Paper Scissors** game implemented in Python. Play against the computer to test your luck and strategy!

---

## Features

* **Classic Gameplay:** Follows the standard rules of Rock beats Scissors, Scissors beats Paper, and Paper beats Rock.
* **Flexible Input:** Accepts various spellings and abbreviations for **Rock**, **Paper**, or **Scissors** (e.g., 'R', 'rock', 'Scissor', 'Pap').

---

## How to Run

### Prerequisites
* **Python 3.x**

### Execution
1.  Save the code as a Python file (e.g., `main.py`).
2.  Open your terminal or command prompt.
3.  Navigate to the directory where you saved the file.
4.  Run the game using the following command:

    ```zsh
    python main.py
    ```

5.  Follow the prompt to enter your choice.
